# 墨 — Тренажёр каны

Минималистичный браузерный тренажёр для изучения **хираганы** и **катаканы** с интервальным повторением, озвучкой японского произношения и подробной статистикой занятий.

Проект работает полностью на стороне браузера: не требует регистрации, базы данных, серверной части или сборки. Прогресс, настройки и история занятий сохраняются локально на устройстве пользователя.

## Возможности

- **208 карточек** хираганы и катаканы:
  - 92 базовых знака;
  - 50 знаков с дакутэном и хандакутэном;
  - 66 сочетаний ёон.
- Выбор набора: **хирагана**, **катакана** или обе азбуки одновременно.
- Дополнительное подключение озвончённых знаков и сочетаний ёон.
- Три режима ответа:
  - ввод ромадзи с клавиатуры;
  - выбор одного из четырёх вариантов;
  - смешанный режим.
- Поддержка распространённых вариантов романизации: например, `shi` / `si`, `chi` / `ti`, `tsu` / `tu`.
- Интервальное повторение по модели кривой забывания Эббингауза.
- Приоритет новых, просроченных и сложных карточек.
- Обязательный возврат ошибочного знака через несколько вопросов.
- Защита от частого показа одного и того же знака подряд.
- Карта знаний с цветовой индикацией прогресса от ошибок к уверенным ответам.
- Японское произношение через Web Speech API.
- Отдельные звуковые сигналы для правильного и неправильного ответа.
- Светлая и тёмная темы с сохранением выбранного оформления.
- Адаптивный интерфейс для компьютеров, планшетов и смартфонов.
- Управление с клавиатуры и поддержка основных ARIA-атрибутов доступности.

## Как работает повторение

Для каждой карточки рассчитывается вероятность воспоминания:

```text
R = e^(-t / S)
```

где:

- `R` — текущая вероятность вспомнить знак;
- `t` — время после последнего повторения;
- `S` — индивидуальная устойчивость памяти для карточки.

Целевая вероятность удержания установлена на **85%**. Когда расчётное значение опускается ниже этого уровня, карточка попадает в очередь повторения.

После правильного ответа устойчивость памяти увеличивается, а следующий интервал постепенно растёт от минут до часов, дней и месяцев. Слишком ранний повтор увеличивает интервал осторожно, а успешное воспоминание после заметного забывания усиливает его сильнее.

После ошибки:

1. устойчивость карточки уменьшается;
2. знак получает повышенный приоритет;
3. карточка возвращается через 2–4 других ответа;
4. знак остаётся в текущей сессии, пока пользователь не ответит правильно.

Первый успешный интервал составляет около **20 минут**. Карточка считается освоенной, когда рассчитанный интервал достигает **21 дня**. Максимальный интервал ограничен **180 днями**.

Когда доступных карточек больше нет, тренажёр показывает время до ближайшего повторения и автоматически возобновляет сессию после наступления срока.

## Статистика

Во вкладке «Статистика» отображаются:

- количество ответов и повторений за сегодня;
- примерное время занятий и среднее время ответа;
- календарь активности за выбранный год;
- среднее число ответов за учебный день;
- процент дней с занятиями;
- текущая и лучшая серия учебных дней;
- общая точность ответов;
- средняя расчётная сохранность материала в памяти;
- количество карточек к повторению в ближайшие 7 и 30 дней;
- прогноз повторений на следующие семь дней;
- общее количество ответов.

В основном экране отдельно показываются карточки:

- **к повторению**;
- **в процессе изучения**;
- **освоенные**;
- точность текущей сессии.

## Карта знаний

Каждый знак отображается отдельной ячейкой:

- серый — знак ещё не изучался или результаты нейтральны;
- красный — преобладают ошибки;
- зелёный — преобладают правильные ответы.

При наведении на показатели прогресса карта временно фильтруется по выбранной категории. Нажатие на знак запускает его японское произношение.

## Хранение данных и приватность

Приложение не отправляет учебные данные на сервер. Вся информация хранится в `localStorage` текущего браузера:

```text
kanaTrainerProgressV1  — прогресс и расписание карточек
kanaTrainerSettingsV1  — выбранный набор, режим, тема и звук
kanaTrainerHistoryV1   — ежедневная история занятий
```

Это означает, что:

- прогресс сохраняется после закрытия вкладки;
- данные привязаны к конкретному браузеру и устройству;
- очистка данных сайта удалит сохранённый прогресс;
- синхронизации между устройствами пока нет.

Кнопка «Сбросить прогресс» удаляет статистику, расписание повторений и результаты всех карточек после подтверждения пользователя.

### Простой способ

Скачайте репозиторий и откройте файл `index.html` в современном браузере.

Рекомендуемый вариант:

```bash
git clone <URL-РЕПОЗИТОРИЯ>
cd <ПАПКА-РЕПОЗИТОРИЯ>
python -m http.server 8000
```

На Windows также можно использовать:

```bash
py -m http.server 8000
```

```text
http://localhost:8000
```

## Управление

- `Enter` — проверить введённый ответ;
- `1`–`4` — выбрать вариант ответа в режиме теста;
- `←` / `→` — переключиться между вкладками тренировки и статистики, когда вкладка находится в фокусе;
- `Esc` — закрыть окно справки;
- кнопка `♪` — произнести текущий знак.

## Совместимость

Приложение рассчитано на современные версии Chrome, Edge, Firefox и Safari.

Озвучка зависит от поддержки Web Speech API и наличия японского голоса `ja-JP` в браузере или операционной системе. Если подходящий голос недоступен, основные функции тренажёра продолжат работать без произношения. Звуковые сигналы также могут быть заблокированы браузером до первого действия пользователя.

## Особенности реализации

- интерфейс учитывает системную настройку `prefers-color-scheme`;
- анимации сокращаются при включённом `prefers-reduced-motion`;
- очередь использует взвешенный случайный выбор, чтобы чаще показывать новые, просроченные и проблемные знаки;
- недавние карточки получают штраф веса, уменьшающий повторения подряд;
- старые данные прогресса автоматически переводятся на актуальную модель памяти;
- история активности хранится отдельно от состояния карточек.
